Glosario

CPython :es la implementación del lenguaje de programación Python, como se distribuye en python.org. El término «CPython» es usado cuando es necesario diferenciar esta implementación de Jython o IronPython.

División entera:es una división matemática que se redondea hacia el entero menor más cercano. Usa el operador //. Por ejemplo, la expresión 17 // 5 evalúa 3 a diferencia del 3.4 tipo float.

Expresión:es una construcción sintáctica que puede ser evaluada, hasta dar un valor. Es una acumulación de literales, variables, accesos a atributos, operadores o llamadas a funciones.

Función:es una serie de sentencias que retornan un valor al que las llama. También se le puede pasar cero o más argumentos para ser usados en la ejecución de la misma.

IDLE: es el entorno integrado de desarrollo de Python, o Integrated Development Environment for Python. IDLE es un editor básico y un entorno de intérprete que se incluye con la distribución estándar de Python.

Importar: es el proceso mediante el cual el código Python dentro de un módulo, se hace alcanzable desde el programa principal.

Interactivo: Python tiene un intérprete interactivo, lo que significa que puede ingresar sentencias y expresiones en el prompt del intérprete, ejecutarlos de inmediato y ver sus resultados.

Interpretado: Python es un lenguaje interpretado. Esto significa que los archivos fuente pueden ser ejecutados directamente y al momento de presentarse un error, el intérprete se detiene y emite un mensaje.

Módulo: es un objeto que sirve como unidad para organización del código. Los módulos son cargados por el proceso de import.

PEP: Propuesta de mejora de Python (Python Enhancement Proposal). Un PEP es un documento de diseño que brinda información a la comunidad, o describe una nueva capacidad para Python, sus procesos o entorno.

Sentencia: una sentencia es parte de un bloque de código.

Saltos de líneas universales: una interpretación para salto de línea. Para el código Python un fin de línea es '\n'

Zen de Python: es un listado de los principios básicos del lenguaje Python, son útiles para entender y usar el lenguaje. El listado se puede encontrar digitando «import this» en la consola interactiva.
